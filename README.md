# CodeLab: FSD Lab Micro Project
### **Title**: Create a Coding Platform using Angular JS

CodeLab is a neo-brutalist "Hacker Console" platform designed for developer growth, featuring a custom IDE, real-time code execution, and a centralized academic repository.

## 🛠️ Tech Stack
- **Frontend**: Angular 21 (Standalone Components, SSR, Prerender)
- **Styling**: Tailwind CSS (Neo-brutalist Design System)
- **Backend/DB**: Firebase (Firestore Authentication & Storage)
- **Execution Engine**: [onlinecompiler.io](https://onlinecompiler.io) API
- **IDE Engine**: Monaco Editor (VS Code core)

## 🚀 Features
- **Hacker Sidebar**: Unified navigation for Dashboard, Challenges, Submissions, and Leaderboard.
- **Project Documentation**: Built-in `/docs` module with academic and structural details.
- **Dynamic IDE**: Real-time syntax highlighting and code execution in Python, C++, Java, and JS.
- **Vercel Optimized**: Automated routing and SSR support for seamless direct navigation.

## 💻 Local Development
1. **Clone & Install**:
   ```bash
   git clone https://github.com/Iamjunade/fsd2.git
   npm install
   ```
2. **Environment Setup**:
   - Ensure `GEMINI_API_KEY` is set in AI Studio or local environment.
   - Code execution uses the integrated `onlinecompiler.io` API.
3. **Run**:
   ```bash
   npm run dev
   ```
